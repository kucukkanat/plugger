# @plugger/vue

Vue 3 bindings — `providePlugger`, the `PluggerSlot` component, and reactive composables.

Part of [Plugger](../../README.md) — a browser-native meta framework for building
plugin systems for any SPA.

```bash
npm install @plugger/vue @plugger/core
```

See the [documentation](../../README.md#documentation) for usage and live
playgrounds.
